# Kaplan-Meier Survival Curve — Example

Survival analysis of mice vaccinated with an iron uptake system vaccine, generated by three AI workflows using the same data and comparable prompts.

## Dataset

[`data/Joseph Mice.csv`](data/Joseph%20Mice.csv) — 26 mice, 3 columns (Time, Event, Treatment).

| Group | n | Deaths | Censored (day 18) |
|-------|---|--------|--------------------|
| GST   | 10 | 7    | 3                  |
| Vac   | 16 | 5    | 11                 |

Time is in days post-vaccination (0–18). Event = 1 is death; Event = 0 is censored (survived to end of study).

## Starting prompt

All three workflows received the same initial request:

> Create a figure from Joseph_Mice.csv. Kaplan-Meier plot. We are looking at survival of two groups of mice (GST and Vac) since receiving vaccine. Time is in days. Event 1 is death, event 2 is survival to the end. x-axis should go from time = 0 to 18. y-axis reflects the number of mice remaining. Statistical test should be Kaplan-Meier estimator. Use Nature-style formatting.

## Iteration summary

Each workflow went through three rounds. The revision requests were similar across workflows, converging on the same final specifications.

| Version | Changes requested |
|---------|-------------------|
| **v1** | Initial figure from the prompt above |
| **v2** | Add title ("Mice Survival Curve: Iron Uptake System Vaccine"). GST blue, Vac red. Decrease line thickness. Update axis labels and tick intervals (x every 4, y every 5). Move legend to lower left with grey border. Add/reposition log-rank p-value |
| **v3** | 4:3 aspect ratio. GST #0000FF, Vac #FF0000. Remove zero ticks. Fine-tune font sizes (title, axes, legend, stats). Pin legend lower-left corner to data coordinate (1 day, 1 mouse). Re-add statistics |
